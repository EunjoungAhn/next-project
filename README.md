# next.js Manual SetUP 프로젝트 만들어보기

1. npm init -y 로 패키지 설치  
2. npm i next react reac-dom 설치
3. 패키지 설정을 해줍니다.
```javascript
  "scripts": {
    "dev": "next dev",
    "build": "next build",
    "start": "next start"
  },
```
4. pages라는 폴더 추가해야
5. npm run dev가 실행 됩니다.
6. npm run build를 해야 npm start가 가능합니다.
